The DaveKing Skin

by Dave King

Overview

This skin is an adaptation of the Kubrick skin that Michael Heilemann developed for WordPress. Kubrick originally was ported to NucleusCMS by hcgtv.

DaveKing has the same basic look and feel as Kubrick but I have redone it using <table> layout rather than <div> layout. I did this so that the page would resize smoothly as the browser screen is resized or the font size is changed. With <div> layout that Kubrick uses the text would flow outside the boundaries of the boxes that contained it. The other major change is the use of a calendar for archive navigation.

I developed this skin for a simple personal diary blog that I do not share with anyone so there's no fancy links or navigation features. There is a clickable area in the gray border just above the top edge of the white page that will jump you to the new item entry form that's part of the admin interface. You will have to login after clicking this if you are not logged in already.

Installation

  1. Unpack the daveking.zip file into your NucleusCMS "skins/" directory. A new directory called "daveking/" will be created under your "skins/" directory.
  2. If you do not already have the NP_Calendar plugin installed, copy the "skins/daveking/NP_Calendar.php" file to your "nucleus/plugins/" directory and use the "Management -> Plugins -> Install New Plugin" function in the Nucleus admin interface to install the plugin.
  3. Use the "Layout -> Skins -> Import/Export -> Import" function in the Nucleus admin interface to install the "daveking" skin from the drop down list.
  4. Use the "Settings" function under your blog to change the skin to the daveking skin.
